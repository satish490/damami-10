--- README  -------------------------------------------------------------

Tab Tamer is an administration utility that provides easy re-ordering, hiding, and deactivation of tabs and subtabs. You can also rename tab labels.

--- INSTALLATION and USAGE --------------------------------------------------------

1 - use composer require 'drupal/tabtamer'
2 - on any page where you can see a tabs, now you can see new tab "Add tabtamer"
3 - or you can go to /admin/structure/tab_tamer/add
4 - you have to provide your route name, then everything will be magically loaded by ajax.
